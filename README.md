# LOFERSIL Landing Page

This is the official landing page for LOFERSIL, a Portuguese stationery store.

## Documentation

For full documentation, see [docs/README.md](docs/README.md).

## Quick Start

```bash
npm install
npm run build
npm start
```

## Project Structure

- `src/` - Source code
- `api/` - Serverless functions
- `assets/` - Static assets
- `docs/` - Documentation and project management
- `config/` - Configuration files
- `tools/` - Build scripts and utilities
- `tests/` - Test files
